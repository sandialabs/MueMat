Debugging

  • Basic usage of the MATLAB debbuger: keyboard, dbstack, dbquit.
  • To see all variables (private and public) in a class foo, use the command
        >> openvar foo
  • To put MATLAB automatically in debug mode when a run-time error occurs, use the command
       >> dbstop if error;
  • To turn on memory stats for the Matlab profiler, use the command
       >> profile -memory on;
