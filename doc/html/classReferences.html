Class Reference

This is a reference guide for the main classes in MueMat.

  • Operator class Classes to define MueMat's notion of a matrix.
  • Multivector class Classes to define MueMat's notion of a multivector.
  • Multigrid Hierarchy Classes for creating and maintaining the AMG hierarchy itself.
  • Grid Transfers Operators Classes for creating and maintaining grid transfer matrices.
  • Smoothers Classes for creating and managing smoothers.
  • Aggregation Main classes for creating and managing coarsening.
  • Map Main classes for creating and managing Maps.


Published with MATLAB® 7.11
